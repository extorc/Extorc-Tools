# Extorc-Tools
This is an addon i have been working on for a few days now<br>
I dont have much Blender API experience but i love the way this addon is coming out to be

# So the features :
It can Perform Smooth Op
It can Perform Subsurf
It can perform Difference, Union, Intersection and Slice
You can transfer the Origin to Geometry, Center of Mass, Center of Volume and To 3D Cursor

# To Use : 
Download the file
In The scripting workspace in the Text editor , open the SmoothOp.py and click the run button<br>
Control D for using the Menu
